# projet-de-d-materialisation-de-document
Il s'agit ici de la réalisation du projet pour une démarche transformation qui est la numérisation des documents. En ce sens, la première partie de l’application consiste à récupérer l’image, implémenter l’algorithme pour l’OCR (pour la reconnaissance des caractères) afin d’en extraire le contenu c’est-à-dire transforme les images brutes en un format de texte numérique. Ensuite, la deuxième partie de l’application à analyser le résultat obtenu après extraction des données où nous avons tout d’abord fait la classification de texte, nous avons travaillé avec le classifieur Naïve Bayes pour catégoriser nos documents pour traiter chaque document selon sa catégorie puis, structurer nos données en un document de format csv en utilisant des expressions régulières, car c’est la méthode qui nous rapproche du résultat souhaité. ce travail peut être améliorer pour un meilleur résultat.
